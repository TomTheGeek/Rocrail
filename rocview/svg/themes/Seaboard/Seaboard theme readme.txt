Seaboard System theme from Railroad Signals of the US website based on SpDrS60 theme. The following signal masts are supported:-

Prefix	Mast type
	Single head mast automatic
1a_	Single head mast standard head aspect numbers
1_	Single head mast unified Seaboard aspect numbers
2_	Twin head mast unified Seaboard aspect numbers
3_	Triple head mast unified Seaboard aspect numbers

Automatic aspect numbers
-r	R292	red
-y	R285	yellow
-g	R281	green
-w	R290	lunar
-b	-	unlit
